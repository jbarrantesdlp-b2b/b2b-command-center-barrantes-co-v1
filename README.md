# B2B Command Center · Barrantes Co. v1.4

Portada actualizada con:
- Foto de centro de control como fondo
- Logo Geosatelital arriba a la derecha

## Uso

1. Descomprima.
2. Copie su `.env` actual dentro de esta carpeta.
3. Ejecute:

```powershell
npm.cmd install
npm.cmd run dev
```

## Publicar

```powershell
vercel.cmd --prod
```
